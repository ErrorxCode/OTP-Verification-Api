## Implimentation
Add maven to your root build.gradle
```
allprojects {
	repositories {
		...
		maven { url 'https://jitpack.io' }
	}
}
```
Add the dependency
```
dependencies {
	    implementation 'com.github.ErrorxCode:OTP-Verification-Api:Tag'
}
```
